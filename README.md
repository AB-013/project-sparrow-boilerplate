## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash

# set up env file
$ cp ./.env.sample ./.env

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Migrations

```bash
# generate new migration
$ npm run migration:generate migration-name

# run migrations
$ npm run migration:run

# revert previous migration
$ npm run migratin:revert
```
## Features
**Basic Authentication:** The project includes basic authentication using JWT and PassportJS.

**Auth Guard and Decorator:** Implement an authentication guard and decorator for securing routes.

**CRUD for Users:** Basic CRUD operations are available for managing user data.

**Swagger**: Swagger is integrated for API documentation. Access the API documentation at http://localhost:3000/docs.

**Prettier and ESLint:** Code formatting and linting are set up for consistent and clean code.

**Husky Hooks:** Git commit and push hooks are configured with Husky for code quality checks.

**Absolute Paths:** Absolute import paths are configured for easier module imports.

**Dockerfile:** A Dockerfile is provided for containerizing your application.

**Tests Setup:** Unit and integration tests are set up for testing your application.

**Commit Message Linting:** Commit messages are linted to follow conventional commit message standards.

**Sample Migrations:** Sample database migrations are provided as examples for extending the database schema.
